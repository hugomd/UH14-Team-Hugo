What is QuickCraft?

QuickCraft lets you roll your own Minecraft server in just a couple of minutes. Simply choose the number of days you want to have the server for, pay for the server time (no profits made), and QuickCraft will create a DigitalOcean VPS with a Minecraft server up and running!

Is that it?

No. I'm planning to develop a web panel to do simple things like restart the server and load custom maps.

What happens after I pay?

Once the payment goes through, you'll receive an email containing the IP address and port of your new Minecraft server!
